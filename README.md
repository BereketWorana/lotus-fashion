# 🪷 Lotus Fashion

An Ethiopian luxury fashion e-commerce frontend built with Next.js, TypeScript, and Framer Motion.

## 🌐 Live Demo
👉 Coming soon

## ✨ Features
- Animated hero section with parallax scrolling
- Product shop with category filtering
- Fully functional cart with quantity controls
- USD / ETB currency toggle
- Custom animated Lotus SVG logo
- Fully responsive design
