# 🪷 Lotus Fashion
 
An Ethiopian luxury fashion e-commerce frontend built with Next.js 14, TypeScript, and Framer Motion.
 
## 🌐 Live Demo
👉 [View Live](https://lotus-fashion.vercel.app)
 
## ✨ Features
 
- Animated hero section with parallax scrolling
- Product shop with category filtering
- Fully functional cart with quantity controls
- USD / ETB currency toggle
- Smooth page transitions with Framer Motion
- Custom animated Lotus SVG logo
- Fully responsive design (mobile + desktop)
- Dark luxury aesthetic inspired by Ethiopian heritage
